## src
Source code for InTAct project.
- `main.py` - Entry point for the project.
- `dataset/` - Custom continual learning datasets missing in Continuum.
- `experiment.py` - Main experiment procedure.
- `model/` - Full models and custom layers.
- `method/` - Continual learning methods.
- `visualization/` - Visualization of how InTAct works.
- `util/` - Utilities.
- `scenario/` - Custom CL scenarios missing in Continuum.
